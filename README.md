## Local Development

This site is built with **Jekyll** and **Materialize CSS**.

### Setup

```bash
bundle install
bundle exec jekyll serve
```

Visit `http://localhost:4000` to view locally.

### Structure

- `index.html` — Landing page with sections
- `about.md` — About page
- `_config.yml` — Site configuration
- `_layouts/` — Page templates
- `_includes/` — Reusable components
- `css/` — Stylesheets
- `js/` — JavaScript
